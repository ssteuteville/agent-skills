# General Agent Skills

Domain-agnostic skills for structured thinking, brainstorming, and problem-solving workflows.

## Prerequisites

None — these skills rely only on conversational techniques and don't require external tools.

## Skills

### [how-might-we](how-might-we/)

Facilitates "How Might We?" brainstorming sessions for software engineering challenges. Guides the user through reframing a problem as HMW questions, brainstorming solutions for each, and produces a structured summary document at the end.
